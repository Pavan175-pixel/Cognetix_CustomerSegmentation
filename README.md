# Customer Segmentation Using K-Means Clustering

## Objective
Group customers into meaningful segments based on Annual Income and Spending Score using K-Means Clustering.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Project Workflow
1. Data Loading
2. Data Exploration
3. Data Visualization
4. Feature Selection
5. Elbow Method
6. K-Means Clustering
7. Cluster Visualization
8. Silhouette Score Evaluation

## Features Used
- Annual Income (k$)
- Spending Score (1-100)

## Outputs
- Customer Segments
- Cluster Visualization
- Elbow Method Graph
- Silhouette Score

## How to Run
1. Install requirements
2. Open customer_segmentation.ipynb
3. Run all cells
